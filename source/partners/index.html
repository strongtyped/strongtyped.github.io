Typesafe partnership

Being a Typesafe consulting partner, Strong[Typed]

  • organizes official Typesafe training on Scala, Akka and Play. The training is given by Typesafe trainers
  • organizes official Typesafe consulting services on the Typesafe Reactive Platform, consisting of Scala, Akka, Play, Activator and Spark
  • offers its own consulting services on the Typesafe Reactive Platform

Typesafe Reactive Platform

Reactive applications are message driven, responsive, resilient and elastic.

Scala

Scala is a hybrid functional and object-oriented programming language that runs on the Java Virtual Machine. As such, Scala can seamlessly make use of Java libraries.

Akka

Akka is a framework that implements the Actor Model on the Java Virtual Machine. Akka manages concurrency and distribution in a locally transparent way.

Play

Play is a RESTful application framework which is asynchronous and supports push channels.

Spark

Spark is a distributed computing system that is a superior alternative to Hadoop's MapReduce.
